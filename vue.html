Vue 3 – Quick Reference

Grunnleggende verktøy
1. Props data inn
type Props = { value: number; label?: string }
const props = defineProps<Props>()
2. Emits data ut
const emit = defineEmits<{
  (e: 'increment'): void
  (e: 'update:value', value: number): void
}>()
3. Reactivity + computed
// ref -> .value i script
// reactive -> direkte tilgang
const count = ref<number>(0)
const state = reactive({ count: 0, name: '' })
const double = computed(() => count.value * 2)
4. Oppdatere state
const increment = () => {
  count.value++ // ref -> .value
  state.count++ // reactive -> direkte
}
5. Script setup
<script setup lang="ts"></script>
6. Expressions
{{ count }}
{{ count + 1 }}
{{ isActive ? 'Yes' : 'No' }}
7. Binding
<div :id="id"></div>
<div :class="{ active: isActive }"></div>
<div :class="['btn', isActive && 'active']"></div>
<div :style="{ color: 'red' }"></div>
8. Events
<button @click="increment"></button> <!-- kall funksjon -->
<button @click="emit('increment')"></button> <!-- send opp -->
<button @click="count++"></button> <!-- inline endring -->
<button @click="doSomething(id)"></button> <!-- argument -->
<form @submit.prevent="submit"></form> <!-- stopp reload -->
<button @click.stop="click"></button> <!-- stopp bubbling -->
<input @keyup.enter="save" /> <!-- Enter-tast -->
9. v-model
const text = ref('')
<input v-model="text" />
10. Conditionals
<div v-if="isVisible"></div>      <!-- render -->
<div v-else-if="other"></div>
<div v-else></div>
<div v-show="isVisible"></div>    <!-- toggle ofte -->
11. Loops (v-for)
<div v-for="item in items" :key="item.id">{{ item }}</div>
<div v-for="(item, index) in items" :key="index"></div>
<div v-for="(value, keyName) in obj" :key="keyName"></div>
<div v-for="n in 6" :key="n">{{ n }}</div>
12. Korte huskeregler
  • props er readonly – ikke muter dem.
  • emit sender hendelser/data opp til parent.
  • ref: bruk .value i script, ikke i template.
  • computed brukes for avledet state.
  • v-if lager/fjerner DOM. Bruk når noe sjelden vises.
  • v-show skjuler/viser med CSS. Bruk når noe toggles ofte.
  • v-for: bruk alltid :key, helst stabil id.
  • Unngå v-if og v-for på samme element.
